merge sort

对一个无序数组，在每一次迭代中，平均分为2份，之后进行合并。同值元素可保序。

  • Time complexity:
    T(n) <= 2 T(n / 2) + O(n)
    <= 2     (2 T(n / 4) + O(n / 2)) + O(n) = 4 T(n / 4) + 2 O(n)
    <= ……
    <= 2k     T(n / 2k) + k O(n)
    So T(n) <= O(n     log(n))
