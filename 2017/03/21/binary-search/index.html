对有序数组的二分查找

  • Time complexity:
    T(n) <= T(n / 2) + O(1)
    <= (T(n / 4) + O(1)) + O(1) = T(n / 4) + 2 O(1)
    <= ……
    <= T(n / 2k) + k     O(1)
    Let n / 2k = 1, then k = log2n相当于log(n)
    So T(n) <= O(log(n))
